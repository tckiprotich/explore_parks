# Endpoints

### GET /api/getAllParks
Description: Retrieves all parks asynchronously.

##### Example Request:

```http
GET /api/getAllParks
```

##### Example Response:

```json
{
  "success": true,
  "message": "Parks retrieved successfully",
  "data": [ /* Array of ParkModel objects */ ]
}
```

### POST /api/Users
Description: Create a new user

Example Request:

```http
GET /api/Users
```

##### Parameter
```json
{
  "userName": "string",
  "password": "string",
  "email": "string"
}
```

##### Example Response:

```json
{
  "userName": "string",
  "email": "string"
}
```
